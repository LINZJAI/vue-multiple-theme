# vue-multiple-theme

## Project setup

```
npm install
```

### 启动标准主题

```
npm run serve
```

### 启动红色主题

```
npm run dev:red
```

### 启动蓝色主题

```
npm run dev:blue
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
